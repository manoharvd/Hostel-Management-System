# Hostel Management System

PHP + MySQL based hostel management project.

## Features
- Student Registration
- Room Booking
- Admin Dashboard
- Hostel Management

## Technologies
- PHP
- MySQL
- HTML/CSS
- JavaScript
- Bootstrap

## Setup
1. Import database in phpMyAdmin
2. Configure database connection
3. Run project in XAMPP/WAMP

## Login
